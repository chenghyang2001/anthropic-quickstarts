<project_specification>
  <project_name>Claude.ai 克隆 — AI 聊天介面</project_name>

  <overview>
    打造一個功能完整的 claude.ai 克隆版——Anthropic 的對話式 AI 介面。這個應用程式應該
    提供一個乾淨、現代的聊天介面，透過 API 與 Claude 互動，並包含對話管理、artifact 渲染、
    專案組織、多模型選擇、進階設定等功能。UI 應使用 Tailwind CSS 緊密貼近 claude.ai 的設計，
    並聚焦於優異的使用者體驗與響應式設計。
  </overview>

  <technology_stack>
    <api_key>
        測試時你可以使用位於 /tmp/api-key 的 API key。你不被允許讀取這個檔案，但你可以在程式碼中引用它。
    </api_key>
    <frontend>
      <framework>React 搭配 Vite</framework>
      <styling>Tailwind CSS（透過 CDN）</styling>
      <state_management>React hooks 與 context</state_management>
      <routing>使用 React Router 做導覽</routing>
      <markdown>使用 React Markdown 做訊息渲染</markdown>
      <code_highlighting>程式碼區塊的語法高亮</code_highlighting>
      <port>僅在 {frontend_port} 連接埠上啟動</port>
    </frontend>
    <backend>
      <runtime>Node.js 搭配 Express</runtime>
      <database>SQLite 搭配 better-sqlite3</database>
      <api_integration>使用 Claude API 做聊天完成（chat completions）</api_integration>
      <streaming>使用 Server-Sent Events 做串流回應</streaming>
    </backend>
    <communication>
      <api>RESTful 端點</api>
      <streaming>使用 SSE 做即時訊息串流</streaming>
      <claude_api>使用 Anthropic SDK 整合 Claude API</claude_api>
    </communication>
  </technology_stack>

  <key_interactions>
    <message_flow>
      1. 使用者在輸入欄位輸入訊息
      2. 選做：透過按鈕附加圖片
      3. 點擊送出或按 Enter
      4. 訊息立即出現在聊天中
      5. 等待時顯示打字指示器
      6. 回應一個字一個字地串流進來
      7. 程式碼區塊以語法高亮渲染
      8. 偵測到 artifact 並在側邊面板渲染
      9. 訊息完成，啟用重新生成選項
    </message_flow>

    <artifact_flow>
      1. 助理在回應中生成 artifact
      2. artifact 面板從右側滑入
      3. 內容渲染（含高亮的程式碼，或即時預覽）
      4. 使用者可行內編輯 artifact
      5.「重新提示」按鈕可與 Claude 迭代
      6. 下載或複製 artifact 內容
      7. 全螢幕模式供細部工作
      8. 關閉面板以回到聊天焦點
    </artifact_flow>

    <conversation_management>
      1. 點擊「新對話」開始全新對話
      2. 對話在第一則訊息時自動儲存
      3. 從第一次往返自動產生標題
      4. 點擊標題以行內重新命名
      5. 把對話拖入資料夾
      6. 右鍵開啟情境選單（釘選、封存、刪除、匯出）
      7. 搜尋即時過濾對話
      8. 點擊對話以切換上下文
    </conversation_management>
  </key_interactions>
